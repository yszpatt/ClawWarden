# 更新日志 (CHANGELOG)

本文档记录了 ClawWarden 项目的主要更新内容。

---

## [2026-02-04]

### ✨ 新特性 (Features)
- **增量任务摘要系统**: 重构了任务摘要逻辑，现在支持增量更新，并采用层次化的折叠式（Accordion）视图展示任务进度。
- **对话式交互模型**: 强化了原生聊天界面，将其作为任务执行过程中的主要指令下达和反馈入口。

### 💄 UI/UX 优化 (Major Overhaul)
- **外观重设计**: 进行了大规模的 UI 视觉升级，引入了更深邃的毛玻璃效果、优化的深色模式调色板以及更现代的排版。
- **任务详情页重构**: 完全重写了任务详情面板，优化了信息层次结构，并提升了在大数据量下的渲染性能。
- **泳道视觉同步**: 为看板的每个阶段（设计、开发、测试等）同步了专属的主题色、图标和语义化的操作按钮（如“开始自动化测试”、“合并到主分支”）。

### 🗑️ 功能精简 (Cleanup)
- **移除“原始输出”标签页**: 彻底移除了复杂且低效的 Xterm.js 终端组件及后端流式日志，转而通过更易读的对话和摘要来呈现结果。
- **依赖优化**: 移除了 `xterm`、`xterm-addon-fit` 和 `node-pty` 等重型依赖，显著减轻了安装包体积和运行负担。

### 📝 文档 (Documentation)
- 同步更新了中英文 README，反映了最新的功能变化和技术栈更新。

---

## [2026-02-03]

### ✨ 项目管理优化
- **增强型项目选择器**: 实现了基于本地文件夹的选择、移除以及自动导入功能，项目管理更加直观。
- **界面可见性**: 修复了深色模式下任务详情和选项卡的文字对比度问题。

---

## [2026-02-02]

### 🏗️ 架构与品牌重塑
- **项目更名**: 正式从 "AntiWarden" 更名为 **"ClawWarden"** (Claude Code CLI Warden)。
- **配置迁移**: 配置文件目录从 `.antiwarden` 迁移至 `.clawwarden`。
- **数据持久化**: 增强了 Worktree 历史追踪和数据持久化机制。

### 💬 对话系统增强
- **对话显示升级**: 重新设计了对话消息列表，支持交错显示消息块（Interleaved Blocks），提升了阅读长对话的体验。
- **状态追踪**: 优化了工具调用（Tool Call）的状态追踪和消息处理逻辑。

---

## [2026-01-31]

### 🚀 核心集成
- **全链路集成**: 实现了设计（Design）、开发（Develop）、测试（Test）泳道与流式对话系统的完整闭环。
- **API 统一**: 统一了提示词（Prompt）处理逻辑，移除了过时的设计路由。

---

## [2026-01-30]

### ⭐ 基础功能
- **对话存储服务**: 在后端集成了对话存储与 API 路由，为前端提供会话持久化支持。
- **共享模型**: 定义了跨端共享的对话相关 TypeScript 类型。

- **Release v0.1.0**: 发布首个稳定的初始版本，包含看板基础功能与 Claude Code 初步集成。
